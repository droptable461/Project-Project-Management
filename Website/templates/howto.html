PPM How to Use the PPM Website

The login page is linked to our database which holds only the usernames of the people in the DROPTABLE 461 group. So you can use allie, breonna, thomas, tejas, brandon, and dakota as a username to login to the app

The HOME page is used to store a table of all of the users that are in the DROPTABLE 461 senior sem group. It gives an explanation of what each users job was throughout the entire project.

The TIMELINE page shows all of the updated work that has been done throughout the time we started the project. This page does not show who worked and updated what it just shows the tasks that were set at the time.

This page is where the user logged in can create a project with any title, write who is manager of the project and create a description for it. The projects that are submitted will be added to the sidenav bar once the page is refreshed. They can also add a task into any (to do, doing, done) phase. They can also add a new column and change the title of the column.

On the MY PROJECTS page when adding in a new project, after hitting the submit button you need to refresh the page before the new project will show up on the side bar.
